# Becoming Me

**Becoming Me** is a pixel-style interactive narrative game about queer/trans growth across different stages of life.  
The project explores how identity, safety, and support are shaped by family, school, digital space, work, and medical institutions.

## Project Concept

The game is structured as a life timeline with five stages:

1. **Childhood** – a bedroom filled with early feelings of difference  
2. **Adolescence** – a classroom shaped by secrecy, peer pressure, and private writing  
3. **Online** – profile-making, visibility, messages, and algorithmic discovery  
4. **Job Search** – professionalism, self-presentation, and institutional pressure  
5. **Clinic** – care, bureaucracy, documentation, and recognition  

Players click objects in each panel to trigger narrative text and make choices.  
Each choice changes three values:

- **Identity**
- **Safety**
- **Support**

These values shape the final ending.

## Features

- Pixel-art visual storytelling
- Clickable interactive hotspots in each stage
- Choice-based narrative responses
- Dynamic stat system
- Multiple endings
- Panel-specific exploration progress
- Reset button for each panel
